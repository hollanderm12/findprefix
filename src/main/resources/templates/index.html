Find Prefix

This program searches a specified prefix and outputs, from a list of the 56000 most common English words, all words containing that prefix in JSON format using REST.

Enter a string below, then click Search. Note that British English spelling is used (e.g. "colour" and "theatre" instead of "color" and "theater") and certain phrases are included (e.g. "HAPPYGOLUCKY").
